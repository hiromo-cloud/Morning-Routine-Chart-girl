# 🎀 あさのしたくボード (Morning Routine App)

お子様の朝の準備を視覚的にサポートし、「自分でできた！」という達成感を育むためのルーチン管理アプリです。

## ✨ 特徴
- **視覚的なタイマー**: 残り時間を円形ゲージで表示し、時間の感覚を直感的に伝えます。
- **スタンプカード機能**: 全てのタスクを完了すると、一週間分のスタンプが貯まります。
- **カスタマイズ可能**: お子様のペースに合わせて、タスクの内容や目標時間を「おとなの設定」から変更できます。
- **Firebase 連携**: データの永続化により、デバイスを変えても進捗を維持できます。

## 🛠 技術スタック
- **Frontend**: React 18 / Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React / Custom Inline SVG
- **Backend**: Firebase (Auth / Firestore)
- **Deployment**: Vercel
